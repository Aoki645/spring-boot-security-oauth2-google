## Microsevice with Spring Boot Security OAuth2 Google and Angular JS

### Technology stack:

* Maven;
* FindBugs;
* Travis CI;
* Tomcat embedded;
* Spring Boot;
* JUnit
* Logback
* SLF4J
* Spring MVC;
* Spring Data JPA;
* Hibernate (as JPA implementation);
* MySQL Relation Database;
* Spring Security (as basic authentication);
* Angular JS, HTML, CSS.

### To run this application use:

```bash
mvn spring-boot:run
  ```
